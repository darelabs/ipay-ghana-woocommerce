iPay Ghana WooCommerce is a WordPress plugin for your [WooCommerce](https://woocommerce.com/) store. It enables you receive payments online in Ghana, from your customers.

== Description ==

iPay Ghana WooCommerce is a WordPress eCommerce plugin that enables you to receive payments online in Ghana, from your customers. Built on WooCommerce, it is easy to configure and use.

Visit our [get started](https://manage.ipaygh.com/xmanage/get-started) page and sign up with the easy steps provided.

== Installation ==

= Minimum Requirements =
* WooCommerce plugin version 3.0.1 or greater
* iPay Ghana Merchant account (login or sign up [here](https://manage.ipaygh.com/xmanage/get-started) to get started)
* PHP version 5.2.4 or greater (PHP 7.2 or greater is recommended)
* MySQL version 5.0 or greater (MySQL 5.6 or greater is recommended)
* You should also check [WordPress requirements](http://wordpress.org/about/requirements/) and [WooCommerce requirements](https://docs.woocommerce.com/document/server-requirements/)

= Automatic installation =

To install the iPay Ghana WooCommerce plugin with much ease and directly from your WordPress dashboard,
1. Go to Plugins > Add New.
2. Type "iPay Ghana WooCommerce" in the search box provided; from the search result, click on the "Install Now" button on the iPay Ghana WooCommerce plugin card to install it.
3. After installation you can configure and enable the iPay Ghana WooCommerce plugin under your WooCommerce settings dashboard.

= Manual installation =

To [manually install](https://codex.wordpress.org/Managing_Plugins#Manual_Plugin_Installation) the iPay Ghana WooCommerce plugin, you must first download it, then, unzip and upload it to the `/wp-content/plugins/` directory of your webserver using your favourite FTP application.

After this stage, proceed to the Plugins page of your WordPress dashboard so as to locate and Activate the iPay Ghana WooCommerce plugin.

You may now visit your WooCommerce Settings page and configure the iPay Ghana WooCommerce plugin based on your preferences.
